O que é um UPDATE?

O comando UPDATE SET, que significa "atualizar", é usado para modificar registros já existentes em uma tabela do BD. Geralmente usamos o WHERE junto com o UPDATE para evitar alterações em MASSA não planejadas.

Sintaxe de um UPDATE em SQL
UPDATE nome_da_tabela
SET nome_da_coluna1 = valor1, nome_da_coluna2 = valor2, ...
WHERE condição_desejada;

No exemplo abaixo a tabela se chama CLIENTES e as colunas que serão alteradas são "UFcliente" e "cidadeCliente". Antes de executar o UPDATE, faça um SELECT na tabela CLIENTES e verifique os valores do Cliente com id = 1.

UPDATE CLIENTES
SET cidadeCliente = 'Guarapari', UFCliente = 'ES'
WHERE idCliente = 1;

Após executar o UPDATE, faça outra vez o SELECT e perceba que os campos foram atualizados.

Se não usassemos o WHERE, todos os clientes teriam mudado sua cidade e estado para Guarapari, ES. Não queremos isso em um ambiente real!

Alguns conceitos importantes

Cuidados com UPDATE: Mesmo usando o WHERE, é necessário verificar se existem mais registros atendendo essa condição. Se usassemos WHERE cidadeCliente = 'Lagoa Santa' pois vimos que o cliente 1 é de Lagoa Santa, iriamos mudar o cliente 5 também! Então muito cuidado e pratique aqui antes de usar :).
