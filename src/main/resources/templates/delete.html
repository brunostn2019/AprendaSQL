O que é um DELETE?

O comando DELETE FROM, que significa "deletar", é usado para excluir registros em uma tabela do BD. Geralmente usamos o WHERE junto com o DELETE para evitar exclusão em MASSA, assim como no UPDATE.

Sintaxe de um DELETE em SQL
DELETE FROM nome_da_tabela
WHERE condição_desejada;

No exemplo abaixo a tabela se chama CLIENTES e vamos excluir o cliente com id = 2. Antes de executar o DELETE, faça um SELECT na tabela CLIENTES e verifique quem é o cliente 2.

DELETE FROM CLIENTES
WHERE idCliente = 2;

Após executar o DELETE, faça outra vez o SELECT e perceba que o cliente 2 sumiu.

Se não usassemos o WHERE, todos os clientes teriam sido deletados. Faça esse teste, e não se preocupe, nossa base é refeita sempre que atualizar a página.

Alguns conceitos importantes

DELETE: Apesar do nome, não podemos deletar tabelas ou colunas com este comando. Apenas registros de uma tabela específica. Para deletar uma tabela inteira ou uma coluna existe o comando DROP.
