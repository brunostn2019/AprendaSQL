Como usar o ORDER BY?

O palavra-chave ORDER BY, que significa "ordenar por", é usada para ordenar os registros selecionados em ordem crescente ou decrescente. Podemos adicionar a palavra-chave ASC para usarmos a ordem crescente, para usar a ordem decrescente é necessário usar outra palavra-chave: DESC. Se não adicionar nenhuma dessas palavras-chave, o padrão é por ordem crescente.

Sintaxe do ORDER BY
SELECT nome_da_coluna, nome_da_coluna2, ...
FROM nome_da_tabela
ORDER BY nome_da_coluna,nome_da_coluna2,...;

Nos exemplos abaixo, usar o ASC ou usar nenhum, dá o mesmo resultado (crescente). Vamos supor que queremos ver os clientes em ordem alfabetica, então usamos 'ORDER BY nomeCliente'. Usar o DESC mudará a ordem para decrescente. 

SELECT idCliente, nomeCliente, UFCliente FROM CLIENTES ORDER BY nomeCliente;
SELECT idCliente, nomeCliente, UFCliente FROM CLIENTES ORDER BY nomeCliente ASC;
SELECT idCliente, nomeCliente, UFCliente FROM CLIENTES ORDER BY nomeCliente DESC;

Nos últimos exemplos usamos 2 colunas no ORDER BY, podemos usar quantas forem necessárias. A ordenação é feita na primeira coluna, em seguida, na segunda, e assim por diante. 

SELECT * FROM CLIENTES ORDER BY nomeCliente, UFCliente;
SELECT * FROM CLIENTES ORDER BY UFCliente, nomeCliente;
SELECT * FROM CLIENTES ORDER BY UFCliente ASC, nomeCliente DESC;

Execute os exemplos e perceba as diferenças nos resultados apresentados.
