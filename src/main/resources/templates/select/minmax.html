SQL MIN e MAX

A função MIN retorna o menor valor da coluna selecionada. E a função MAX retorna o maior.

Sintaxe da MIN
SELECT MIN(nome_da_coluna)
FROM nome_da_tabela
Sintaxe da MAX
SELECT MAX(nome_da_coluna)
FROM nome_da_tabela

No exemplo abaixo queremos o menor preço (coluna valorProduto) da tabela PRODUTOS.

SELECT MIN(valorProduto)
FROM PRODUTOS;

Agora queremos saber o maior preço.

SELECT MAX(valorProduto)
FROM PRODUTOS;

Se usarmos essas função em um campo VARCHAR (String) ele retornaria o primeiro nome em ordem alfabética para o MIN e o último nome para o MAX.

Apesar de funcionar em campos STRING essas funções são funções SQL numéricas. Em SQL existem muitas outras funções para vários tipos de dados como string, data, números além de funções como IF ELSE, e etc.
