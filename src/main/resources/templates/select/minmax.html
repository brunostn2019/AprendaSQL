O que é a Cláusula LIMIT?

A cláusula LIMIT em MySQL é usada para limitar a quantidade de registros que serão trazidos pela Query, em outros BDs, a sintaxe para esse tipo de query pode mudar totalmente, por exemplo no SQL SERVER seria equivalente ao "SELECT TOP".

Sintaxe da LIMIT
SELECT nome_da_coluna, nome_da_coluna2, ...
FROM nome_da_tabela
LIMIT numero_de_linhas;

No exemplo abaixo queremos que o resultado mostre apenas uma linha, mesmo que existam mais linhas no resultado.

SELECT *
FROM CLIENTES
LIMIT 1;

Agora queremos saber quem é o último registro adicionado. Podemos só ordenar por ID decrescente e o primeiro resultado é o que queremos certo? Mas vamos supor que existem milhares de registros na tabela PRODUTOS. Dessa forma será mais rápido usar o LIMIT 1.

SELECT *
FROM PRODUTOS ORDER BY idProduto DESC
LIMIT 1;

Podemos colocar qualquer número de linhas após a cláusula LIMIT. Para 10 registros temos que usar 'LIMIT 10'.

Estamos trabalhando em tabelas com poucos registros pois são apenas para nossos estudos. Em um ambiente real podemos ter quantidades enormes de dados. Por isso devemos criar Queries pensando no melhor desempenho possível.

Alguns conceitos importantes

DBA: É sigla para "Database administrator" ou Administrador de Banco de Dados. Este é o profissional que vai cuidar do BD da empresa, tendo grande conhecimento na área, as alterações devem ser analisadas por ele, evitando problemas com segurança e desempenho.
