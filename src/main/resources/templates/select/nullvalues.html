Como usar IS NULL e IS NOT NULL?

O comando IS NULL, que significa "selecionar", é usado para buscar informações em uma tabela do BD (banco de dados). Geralmente é usado com a cláusula FROM seguido do nome da tabela.

Sintaxe de um SELECT em SQL
SELECT nome_da_coluna, nome_da_coluna2, ...
FROM nome_da_tabela;

No exemplo abaixo a tabela se chama "CLIENTES" e as colunas que serão exibidas sao "idCliente" e "nomeCliente". O ponto e vírgula ';' indica o fim da Query, mas não é obrigatório.

SELECT * FROM CLIENTES ORDER BY nomeCliente, UFCliente;
SELECT * FROM CLIENTES ORDER BY UFCliente, nomeCliente;
SELECT * FROM CLIENTES ORDER BY UFCliente ASC, nomeCliente DESC;

Utilize vírgulas para separar os nomes das colunas ou utilize asterisco '*' no lugar do nome das colunas quando quiser trazer todas as colunas existentes na tabela
