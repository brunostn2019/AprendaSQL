O que é SQL?

SQL é a sigla de Structured Query Language, que significa "Linguagem de consulta estruturada". É a linguagem padrão criada para armazenar, manipular e buscar dados em um Banco de Dados.

Exemplo de um SELECT em SQL
SELECT *
FROM CLIENTES

Utilize o código acima no campo indicado e clique no botão "Executar SQL" para realizar sua primeira consulta ao banco de dados

Agora que você já executou seu primeiro comando SQL, navegue pelo menu ou utilize os botões "Anterior" e "Próximo" para conhecer um pouco mais sobre a linguagem SQL.

O nosso objetivo

O objetivo do Aprenda SQL é preparar um curso de SQL em portugês, online e grátis para iniciantes na área de programação, inspirado no W3Schools, onde é possível estudar os conceitos e praticar o que foi apresentado dentro da própria página, sem instalar nada. Ou seja, nosso curso pode ser realizado de qualquer dispositivo que tenha acesso a internet e utilize um navegador compatível com HTML5.
