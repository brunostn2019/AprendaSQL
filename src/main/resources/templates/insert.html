O que é um INSERT?

O comando INSERT INTO, que significa "inserir", é usado para incluir informações em uma tabela do BD (banco de dados). Geralmente nos referimos ao comando como "INSERT" mas a sintaxe correta é INSERT INTO seguido do nome da tabela e os valores desejados conforme o exemplo.

Sintaxe de um INSERT INTO em SQL
INSERT INTO nome_da_tabela (nome_da_coluna1,nome_da_coluna2, ...)
VALUES ('valor1', 'valor2', ...);

INSERT INTO nome_da_tabela
VALUES ('valor1', 'valor2', ...);

No exemplo abaixo a tabela se chama "CLIENTES" e as colunas que serão inseridas sao "nomeCliente" e "cidadeCliente". Antes de executar o INSERT, faça um SELECT na tabela CLIENTES conforme você aprendeu na lição anterior e verifique os campos que ela possui.

INSERT INTO CLIENTES  (nomeCliente, cidadeCliente)
VALUES ('Maria José', 'Brasília');

Após executar o INSERT, faça outra vez o SELECT e perceba que o campo idCliente será preenchido, mesmo não incluindo este campo na Query. Isso ocorre pois o campo idCliente é do tipo INT AUTOINCREMENT ou seja, é um número inteiro que será preenchido automaticamente, adicionando +1 ao último número registrado. O primeiro registro recebe o número 1, o segundo recebe o número 2, e assim por diante.
